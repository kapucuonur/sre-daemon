# TriHonor SRE Daemon
Otonom, kendi kendini iyileştiren (self-healing) SRE-as-a-Service platformu.

## Özellikler
- **Otonom Onarım:** Docker ve sistem servisleri için akıllı hata tespiti ve onarım.
- **Enterprise Ready:** PostgreSQL altyapısı ile yüksek ölçeklenebilirlik.
- **SaaS Kontrol Paneli:** Merkezi API yönetimi, API Key tabanlı müşteri on-boarding.
- **Düşük Gürültü:** Sessiz operasyon, sadece kritik incident raporları.

## Hızlı Kurulum
`curl -sSL https://trihonor.com/install.sh | bash -s -- --key=YOUR_API_KEY`
